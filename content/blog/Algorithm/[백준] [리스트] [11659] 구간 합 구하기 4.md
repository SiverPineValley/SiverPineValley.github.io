리스트를 활용하는 또 다른 방법에는 구간 합이 있다.

리스트를 메모리처럼 활용하는 방법인데, 각 숫자 배열을 입력받을 때 특정 구간까지의 합도 계산해두는 방법이다. 시간 복잡도는 배열을 한 번 순회하는 정도이므로 O(N)이고, 이렇게 합 배열을 만들어두면, 특정 구간 합을 구할 때는 O(1)으로 계산이 가능하다.

먼저, 합 배열 S는 다음과 같이 정의 가능하다.

> S\[i\] = A\[0\] + A\[1\] ... + A\[i-1\] + A\[i\]

보통 여러 숫자를 받아서 리스트에 저장할 것 인데, 이렇게 리스트를 받아놓으면 각 합 배열의 값은 다음과 같이 생성 가능하다.

> S\[i\] = S\[i-1\] + A\[i\]

이렇게 합 배열이 생성되면, 특정 구간 합은 다음과 같이 구할 수 있다.

> \# s ~ e까지의 구간 합  
> S\[s\] + S\[s+1\] + ... + S\[e-1\] + S\[e\] = **S\[e\] - S\[s-1\]**

바로 이런 합 배열을 사용하여 풀이 가능한 문제를 풀어보겠다.

구간 합 구하기4는 N개와 숫자와 M개의 구간이 주어지고, 각 구간들의 합을 출력하면 된다.

다음과 같은 순서로 구현하였다.

> 1\. N, M, l을 입력받음  
> 2\. l을 통해 합 배열 (sum\_l)을 만들기  
> 3\. 각 구간을 입력받으면서 sum\_l을 사용하여 구간 합 출력 (sum\_l\[e\] - sum\_l\[s-1\])

```
"""
    [11659] 평균 구하기
    링크: https://www.acmicpc.net/problem/11659
    카테고리: 자료구조 > 리스트
    날짜: 2026.01.02 15:53PM
    걸린 시간: 15분
"""
import sys
input = sys.stdin.readline

N, M = map(int, input().split())
l = list(map(int, input().split()))
sum_l = [0]
for i, each in enumerate(l):
    sum_l.append(sum_l[i] + each)

for _ in range(M):
    s, e = map(int, input().split())
    print(sum_l[e] - sum_l[s - 1])
```
